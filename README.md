### **One Night Ultimate Werewolf (Web Edition)**

A real-time, multiplayer web implementation of the popular social deduction game One Night Ultimate Werewolf. Built with Node.js, Express, and Socket.io.

# Features
* Real-time Interaction: Synchronized gameplay using WebSockets (Socket.io).

* Role-Specific Interfaces: Unique night-action UI for every role (Seer, Robber, Troublemaker, etc.).

* Dynamic Lobby: Host can choose exactly which roles are in play based on the player count.

* Automated Night Phase: Roles wake up in the correct order as per the official game rules.

* Vote System: Integrated voting phase with automatic winner calculation.

# Tech Stack
* Frontend: HTML5, CSS3, Vanilla JavaScript.

* Backend: Node.js, Express.

* Communication: Socket.io for bi-directional event-based communication.
